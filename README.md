# Co-op Platformer

Repository of a game that features 2D co-op platformer.

## Setup

Refer to Godot's [C# basics guide](https://downloads.godotengine.org/?version=4.6.2&flavor=stable&slug=mono_win64.zip&platform=windows.64) for setting up Godot with C#. Alternatively, reference the instructions below:

- Install [Godot 4.6.2 .NET](https://downloads.godotengine.org/?version=4.6.2&flavor=stable&slug=mono_win64.zip&platform=windows.64).
- Install [.NET SDK](https://dotnet.microsoft.com/download).
- Clone repository locally.
- Configure external editor in Godot settings.
